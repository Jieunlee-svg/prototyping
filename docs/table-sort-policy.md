# 테이블 정렬 정책

## 공통 규칙

| 데이터 타입 | 첫 클릭 기본 방향 | 정렬 방식 |
|---|---|---|
| 일시 (datetime) | 내림차순 ↓ (최신 → 오래된 순) | 문자열 비교 (ISO 형식 보장) |
| 이름 (string) | 오름차순 ↑ (가 → 하) | `localeCompare('ko')` |
| 숫자 (number) | 내림차순 ↓ (높은 → 낮은 순) | 수치 비교 |
| 생년월일 (date) | 내림차순 ↓ (최근생 → 연장자) | 문자열 비교 (ISO 형식 보장) |
| 열거형/상태 (enum) | — | 정렬 불가 |
| 전화번호 | — | 정렬 불가 |
| 액션 버튼 | — | 정렬 불가 |

- 같은 컬럼 재클릭: 방향 토글 (↑ ↔ ↓)
- 다른 컬럼 클릭: 해당 컬럼 기본 방향으로 초기화
- null / 빈 값: 항상 맨 아래
